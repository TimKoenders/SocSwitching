# Local Raw Survey Files

Store restricted original survey files here.

This directory is the raw-provider staging area used by `voteswitchR` and by
the direct CSES Module 6 scripts. It contains election or provider subfolders.
Those subfolders are not necessarily unique datasets: the same CSES module,
European Voter Project file, or national continuity file can be used for
multiple election contexts.

Raw files in this directory are ignored by Git.
